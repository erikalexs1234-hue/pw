**P: Point (Titik Utama)**

Papan tulis tersebut membahas bagaimana **HTML, CSS, dan JavaScript saling terhubung dalam pengembangan web**, dengan penekanan pada **manipulasi DOM (Document Object Model)** menggunakan **Vanilla JavaScript (tanpa framework)** untuk menghasilkan halaman web yang interaktif dan efisien.

---

## **R: Reason (Alasan)**

JavaScript berfungsi sebagai pusat logika atau “otak” dari sebuah website. Pentingnya memahami DOM Selector dan Event Listener adalah karena:

* **Struktur Web**: HTML berperan sebagai kerangka, CSS sebagai tampilan visual, dan JavaScript sebagai pengatur logika.
* **Interaksi**: Agar elemen pada halaman dapat merespons aksi pengguna (seperti klik atau input), JavaScript harus mampu memilih elemen tersebut dan merespons kejadian yang terjadi.
* **Efisiensi**: Pemilihan metode seleksi elemen yang tepat akan memengaruhi kinerja dan kecepatan eksekusi program.

---

## **E: Example (Contoh & Penjelasan Rinci)**

### **DOM Selector (Pemilihan Elemen)**

* **getElementById** digunakan untuk mengambil elemen secara spesifik berdasarkan ID, dan dikenal lebih cepat serta efisien.
* **querySelector** memiliki keunggulan dalam fleksibilitas karena dapat memilih elemen berdasarkan class, ID, maupun tag.

---

### **Event Listener & Function**

Bagian ini menjelaskan bagaimana website dapat memberikan respons terhadap aksi pengguna.

* Menggunakan fungsi sebagai reaksi terhadap suatu kejadian (callback).
* Jenis event yang umum digunakan antara lain: klik, perubahan input, dan fokus pada elemen.

---

### **Output Interaksi**

Untuk memastikan bahwa program berjalan dengan benar, hasil interaksi dapat ditampilkan melalui:

* Tampilan langsung ke pengguna
* Atau melalui alat debugging seperti console

---

### **Manipulasi Data (Value)**

JavaScript juga dapat digunakan untuk mengubah atau mengambil nilai dari suatu elemen, misalnya pada input.
Artinya, data yang dimasukkan pengguna bisa diakses dan dimodifikasi sesuai kebutuhan program.

---

## **P: Point (Kesimpulan)**

Secara keseluruhan, materi ini menegaskan bahwa pemahaman dasar tentang manipulasi DOM menggunakan JavaScript murni sangat penting sebelum mempelajari framework yang lebih kompleks. Dengan menguasai selector, event, dan efisiensi kode, pengembang dapat membuat website yang tidak hanya interaktif tetapi juga memiliki performa yang optimal.
